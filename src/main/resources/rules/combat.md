# Règles de Combat - D&D 5e

## L'Ordre de Marche
Le combat se déroule en rounds. Chaque round dure environ 6 secondes dans le monde du jeu.
1. **Surprise** : Le MJ détermine si quelqu'un est surpris.
2. **Initiative** : Tout le monde lance un dé d'initiative (d20 + Dex).
3. **Tours** : Les participants agissent par ordre décroissant d'initiative.

## Votre Tour
À votre tour, vous pouvez effectuer :
- **Mouvement** : Jusqu'à votre vitesse. Peut être fractionné.
- **Une Action** : Attaquer, Lancer un sort, Foncer, Se désengager, Esquiver, Aider, Se cacher, Se tenir prêt, Chercher, Utiliser un objet.
- **Une Action Bonus** : Seulement si une capacité spéciale le permet.
- **Une Réaction** : Une seule par round (ex: attaque d'opportunité).

## Actions en Combat
- **Attaquer** : Faites un jet d'attaque (d20 + modificateurs) contre la CA de la cible.
- **Lancer un sort** : Temps d'incantation de 1 action.
- **Foncer (Dash)** : Double votre mouvement pour le tour.
- **Se désengager (Disengage)** : Votre mouvement ne provoque pas d'attaques d'opportunité.
- **Esquiver (Dodge)** : Les attaques contre vous ont un désavantage, vos jets de sauvegarde de DEX ont un avantage.
- **Aider (Help)** : Donne l'avantage à un allié pour sa prochaine action.
- **Se cacher (Hide)** : Jet de Discrétion vs Perception passive pour devenir invisible.

## Attaquer
1. **Choisir une cible**.
2. **Déterminer les modificateurs** (Force pour mêlée, Dextérité pour distance/finesse).
3. **Résoudre l'attaque** : d20 + Bonus de Maîtrise (si compétent) + Modificateur de Caractéristique.
   - Si Total >= Classe d'Armure (CA) de la cible, ça touche.
   - **Critique** : Un 20 naturel touche toujours et double les dés de dégâts.
   - **Échec** : Un 1 naturel rate toujours.

## Dégâts et Soins
- **Points de Vie (PV)** : Représentent la durabilité. À 0 PV, vous êtes inconscient.
- **Jets de Sauvegarde contre la Mort** : Au début de votre tour si vous avez 0 PV. 10+ succès, <10 échec. 3 succès = stable, 3 échecs = mort. Un 20 redonne 1 PV. Un 1 compte comme 2 échecs.
- **Dégâts temporaires** : Sont perdus en premier.
